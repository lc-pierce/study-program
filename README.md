studyprogram.py is the main python file. The program is run by:
  1. Adding a database file to the list via the "Add A File..." button
  2. Loading the file into the quiz via the "Load Selections..." button
  3. Launch the quiz

Database files can be created by selecting "Create New..." from the main window and filling put the form fields.

To clear contents of any loaded files, use the "Clear Loaded..." button.
